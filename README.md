# Inventory-Flow-Web
🇧🇷 _Sistema de gestão de inventário focado na transição de lógica CLI para Web. Demonstra a aplicação de conceitos de back-end (CRUD e persistência de dados) em uma interface funcional com JavaScript e LocalStorage. Desenvolvido por Felipe Gonzalez para portfólio técnico._

🇺🇸 Inventory management system focused on transitioning from CLI logic to web-based development. Demonstrating the application of back-end concepts (CRUD and data persistence) in a functional interface using JavaScript and LocalStorage. Developed by Felipe Gonzalez for a technical portfolio.
